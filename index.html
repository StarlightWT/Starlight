Grade calculator

This project was created due to me wanting a simple way to calculate the avg for my grades. It functions on basic js backend and html+css for frontend. It can be found here: https://starlightwt.github.io/GradeCalculator/ the source code can also be found on my github repisotory.

How does it work?
The way it calculates the grade is quite simple, it takes the grade if it has a plus sign next to it the grade's value is .33 better than it, so for example a 2+'s value is acutally 1.66. Then it takes that value and multiplies it by it's weight. The final step is taking all the grades multiplied by their weight and then dividing them by the total weight. It also colours your average in green if it's passing and red if it's not.
